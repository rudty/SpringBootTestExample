# 기타 
### 표준 assert 와 assertThat
- assertThat 은 org.assertj.core.api.Assertions.assertThat
- assert 와 assertThat 모두 사용 가능 
- assert 사용시 간편함 == 비교 true 에만 유의 
- assertThat 은 fail 시 틀린 부분을 보여줌
